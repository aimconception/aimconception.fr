 # AIM Conception - Site Vitrine Professionnel

Bienvenue sur le dépôt public du site internet officiel d'**AIM Conception**, géré par Marius, dessinateur indépendant en bâtiment basé à Lorient (Morbihan). 

Ce site vitrine a pour but de présenter mes prestations de conception de plans de maison, demandes de permis de construire, déclarations préalables de travaux et relevés de l'existant dans les régions du Morbihan et du Finistère.

🌐 **Site en ligne :** [https://www.aimconception.fr](https://www.aimconception.fr)

---

## 🛠️ Technologies utilisées

* **HTML5** : Structuration sémantique avancée, optimisation SEO locale et accessibilité (`aria-labels`).
* **CSS3** : Design moderne, épuré et entièrement responsive (adapté aux mobiles et tablettes).
* **JavaScript (Vanilla)** : Gestion du menu de navigation mobile (burger menu).
* **JSON-LD** : Données structurées intégrées pour un référencement optimal sur Google Maps et Google Search Engine.
